Simply writes a list of possible culprit maps within a text file and exposes them when the server has crashed.

emotes

no_autopickup

NPCMoveTo
Shows a implosion effect when and where a player used a npc_moveto command.

Sven Co-op's PlayerDecalTracker but translated.

Fix multiple errors and/or problems that exists in Sven Co-op and are possible to fix with AngelScript.
CustomPrecache
