DiscordBridge

Bridges and links a Discord server to a Sven Co-op server.

There are three methods to do our bridge.

Open ChatBridge.as and go to the line 19, change the metod for one of the next.



Reads and writtes text files on the game's folder.

No metamod is required.

Requires a discord BOT to be running in the same machine of the server.
Opening files on AngelScript is slow. setting a low value could result on fps drops.



Transfers info directly sending an HTTP Request to a discord's webhook.

Is fast.

Requires metamod.
Only game to discord, no discord to game.



Sends information from one program to another using sockets.

Is fast.

Requires metamod.
Requires a discord BOT to be running in the same machine of the server.


Metamod is optional, you can disable it in ChatBridge.as commenting/removing the line 20.




Configure the json file.
