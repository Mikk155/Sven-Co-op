dynamic_hostname

Change the hostname depending on the map/campaign playing, antirush and survival state

Open dynamic_hostname.json to configure.

CONFIG

HOSTNAME: Name of your server

DISABLED: message to add in $antirush$ and $survival$

ENABLED: message to add in $antirush$ and $survival$

DYNAMIC_HOSTNAME: New name of your server

MAPS

name of the map: title to display

name of the map supports wildcard (*)
