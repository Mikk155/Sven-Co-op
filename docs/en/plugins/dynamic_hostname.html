dynamic_hostname

Change the hostname depending on the map/campaign playing, antirush, survival state and difficulty

Open dynamic_hostname.json to configure.

Original idea by Cubemath

Download 



{
    "CONFIG" :
    {
        // Name of your server
        "HOSTNAME": "Limitless Potential",

        // Text to display when something is disabled
        "DISABLED": "Disabled",

        // Text to display when something is enabled
        "ENABLED": "Enabled",

        // Your new hostname
        // Where $difficulty$ is it will be replaced with the current difficulty of DynamicDifficultyDeluxe
        // Where $antirush$ is it will be replaced if there is any antirush entity available
        // Where $survival$ is it will be replaced if survival mode is active or not
        // Where $map$ is it will be replaced with the "MAPS" category
        "DYNAMIC_HOSTNAME": "$hostname$ | Playing $maps$ | Diff $difficulty$ | Anti-Rush $antirush$ | Survival $survival$"
    },
    // key = name of the current map, supports wildcard (*) at the end or at the start.
    // value = text to display at $map$
    "MAPS":
    {
        "hl_c*": "Half-Life",
        "of*": "Opposing Force"
    }
}
