no_autopickup

Prevent players from automatic pick weapons on touching them

Instead requires players to press Use key (E)

Download 




{
    // true = tell players to press E
    // false = Don't show message
    "MessagePlayer": true,

    // true = Require to be on direct line of sight
    "RequiredLoS": true,

    // List of items this plugin supports. supports ending wildcard as asterisk (*)
    "ItemList":
    [
        "weapon_*",
        "item_healthkit",
        "item_battery",
        "weaponbox",
        "ammo_*"
    ],

    "PickMeMessage":
    {
        "english": "+use pickup $item$",
        "spanish": "+use recoger $item$"
    }
}
