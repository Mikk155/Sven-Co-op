FastRestart

When survival mode is enabled and there is no alive players the map is restarted instantly.

Original idea by Rick.

Download 



json

{
    "ShouldWaitMedic": true,
    "SearchRadius": 1024
}



Configure Json file

graph LR; classDef color fill:black; classDef color color:gray; classDef color align-items:left; ShouldWaitMedic{{"'ShouldWaitMedic' Type: boolean Description: Wait a moment if there are any medic npc around a dead player's body."}}:::color SearchRadius{{"'SearchRadius' Type: integer Description: Radius to search if ShouldWaitMedic is true."}}:::color
