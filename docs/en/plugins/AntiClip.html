AntiClip

Prevent player-to-player clipping and player's projectiles-to-player clipping

Admin command .anticlip (value) to toggle AntiClip.

Possible values are on, true or 1, anything else or empty value is for disable.

Open AntiClip.json to modify the behaviour of projectiles.

Change to false the ones you would like to have collision on ally players.

"lasers" is a special threat, to be exact it is player-player.

If you experiment any collision bugs feel free to disable that.

This plugins requires metamod.
