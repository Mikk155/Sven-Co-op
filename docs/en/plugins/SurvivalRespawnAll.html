SurvivalRespawnAll

When survival is enabled, spawn all players once when they connect for the first time.

Download 



json

{
    "INSTANT_ENABLE": true,
    "SPAWNED":
    {
        "english": "Survival mode now active. Only new players will spawn",
        "spanish": "El modo supervivencia a iniciado. Solo apareceran jugadores nuevos."
    }
}



Configure Json file

graph LR; classDef color fill:black; classDef color color:gray; classDef color align-items:left; INSTANT_ENABLE{{"'INSTANT_ENABLE' Type: boolean Description: If true, nstantly enable survival mode (players will be able to spawn anyways) Else just allow the 'survival will start in & seconds.'-rush"}}:::color
