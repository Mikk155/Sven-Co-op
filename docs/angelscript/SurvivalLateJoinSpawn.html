When survival mode support is enabled the plugin instantly activates survival mode and any player joining the server for a first time during a map will be spawned.

The instant activation of survival mode can be disabled in the json configuration file. 


{
    // Should survival mode activate right away if it's enabled?
    "activate_survival": true,
}

When a player leaves the server while he was still alive. the plugin will keep track of that and spawn them when they reconnect.
