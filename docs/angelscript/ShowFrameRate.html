Allows players to see the server's frame rate information

Adds a client console command ".framerate" to toggle the view.

The info is compossed five values.

1 The current time in seconds since the map started

2 The total count of frames the last second had

3 The current count of frames we're having for this second

4 The current fps count (May fluctuate)

5 Average frame rate per 10 seconds
