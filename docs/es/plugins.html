Plugins

PluginDescripción
AntiClip
Previene la colision entre jugador-jugador y projectiles-jugador
chatroles
Añade cualquier rol a ciertos jugadores, con chatcolors combinado si no se esta utilizando el plugin de Wootguy.
CrashReporter
Simplemente escribe una lista de posibles mapas culpables en un archivo de texto y los expone si el servidor se ha caido.
dynamic_hostname
Cambia el nombre del servidor dinamicamente dependiendo el mapa/campaña, antirush, estado de survival y dificultad
emotes
emotes pero sin 'anal' debido a abusos en la hitbox y ahora cada mensaje esta traducido.
FastRestart
Cuando el modo supervivencia esta activado y no hay mas jugadores vivos el mapa es reiniciado rapidamente.
no_autopickup
Previene que los jugadores recogan los objetos automaticamente tras tocarlos
NPCMoveTo
Muestra un efecto de implosion cuando y donde un jugador utilizo el comando npc_moveto.
PlayerDecalTracker
PlayerDecalTracker de Sven Co-op pero traducido.
SurvivalRespawnAll
Cuando el modo supervivencia esta activo, Permite aparecer a todos los jugadores una vez cuando se conectan por primera vez.

Para plugins por favor utilice en "name" en default_plugins.txt para que sea igual a el nombre del archivo
Ya que estos plugins algunas veces verifican si otros plugins estan instalados.
