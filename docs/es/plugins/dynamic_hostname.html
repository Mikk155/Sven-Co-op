dynamic_hostname

Cambia el nombre del servidor dinamicamente dependiendo el mapa/campaña, antirush, estado de survival y dificultad

Idea original de Cubemath

Descargar 



json

{
    "CONFIG":
    {
        "HOSTNAME": "Limitless Potential",
        "DISABLED": "Disabled",
        "ENABLED": "Enabled",
        "DYNAMIC_HOSTNAME": "$hostname$ | Playing $maps$ | Diff $difficulty$ | Anti-Rush $antirush$ | Survival $survival$"
    },
    "MAPS":
    {
        "hl_c*": "Half-Life",
        "of*": "Opposing Force"
    }
}



Configure el archivo json
