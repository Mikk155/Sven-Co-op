NPCMoveTo

Muestra un efecto de implosion cuando y donde un jugador utilizo el comando npc_moveto.

Este plugin requiere metamod.

Script hecho por Gaftherman.

Descargar 



{
    // true = Observer players can use the visual effect
    // false = only alive players can
    "observers can use": true,

    // Cooldown time, negative value disables this
    // Is not recommended to have low values in here
    "player cooldown": 2.5
}
