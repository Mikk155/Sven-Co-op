SurvivalRespawnAll

Cuando el modo supervivencia esta activo, Permite aparecer a todos los jugadores una vez cuando se conectan por primera vez.

Descargar 



json

{
    "INSTANT_ENABLE": true,
    "SPAWNED":
    {
        "english": "Survival mode now active. Only new players will spawn",
        "spanish": "El modo supervivencia a iniciado. Solo apareceran jugadores nuevos."
    }
}



Configure el archivo json

graph LR; classDef color fill:black; classDef color color:gray; classDef color align-items:left; INSTANT_ENABLE{{"'INSTANT_ENABLE' Tipo: boolean Descripción: Si true, Instantaneamente activa el modo supervivencia (Los jugadores podrán aparecer de todas formas) De otra forma se mantiene el 'Survival will start in & seconds'"}}:::color
