no_autopickup

Previene que los jugadores recogan los objetos automaticamente tras tocarlos

En lugar requiere que los jugadores presionen su tecla Usar (E)

Descargar 




json

{
    "MessagePlayer": true,
    "RequiredLoS": true,
    "ItemList":
    [
        "weapon_*",
        "item_healthkit",
        "item_battery",
        "weaponbox",
        "ammo_*"
    ],
    "PickMeMessage":
    {
        "english": "+use pickup $item$",
        "spanish": "+use recoger $item$"
    }
}
