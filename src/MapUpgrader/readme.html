MapUpgrader

MapUpgrader is a tool to upgrade mods (aka "port") to provide compatibility with Sven Co-op and Seven Kewp as extension.

Usage:

  • Execute MapUpgrader.exe.
  • Choose a script file to execute.
  • Follow the program's instructions.
