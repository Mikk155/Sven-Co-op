GoldSrc2Sven

GoldSrc2Sven is a tool to upgrade mods (aka "port") to provide compatibility with Sven Co-op and Seven Kewp as extension.

Usage:

  • Execute GoldSrc2Sven.exe.
  • Choose a script file to execute.
  • Follow the program's instructions.
